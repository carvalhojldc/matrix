Matrix

For simple dynamic matrix manipulation

About

Matrix is a library that provides simple access to dynamic matrix in C++.

Library

Function list

.getLines() Provides the number of lines in the matrix
.getColumns() Provides the number of columns in the matrix
.newSize(line, column) Sets a new size for the matrix
.inv() Provides a inverse copy of the matrix
.trans() Provides a transposed copy of the matrix

Operations list

== Element-wise equality evaluation of two object
!= Element-wise non-equality evaluation of two objects
= Assign an object or value
+= Addition of two objects
-= Subtraction of one object from another
+ Addition of two objects
- Subtraction of one object from another

Example

Matrix matA(2,2);
Matrix matB(2,2);
Matrix matC;

// Possible access notation
matA(0,0)  = 1;
matA(0,1)  = 2;
matA[1][0] = 3;
matA[1][1] = 4;

// Shows all matrix elements
std::cout << matA;

matA = 0;  // Assigns 0 to all array elements
matB = 10; // Assigns 10 to all array elements

matC = matA * matB;
matA = matB;
matA += matB;

// Changing the size of the matrix
matA.newSize(4,4);
matB.newSize(1,1);
std::cout << matA.getLines() << ' ' << matA.getColumns();
